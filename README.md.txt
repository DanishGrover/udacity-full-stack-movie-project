media.py

 This file contains all the details about movies. 

entertainment_center.py

This file creates instances for  movies. It uses the movie API to gather the movie data and saves the data into an array.

fresh_tomatoes.py

This file contains Javascript, HTML, and CSS scripts that creates .html file showing list of  movies that are created from entertainment_center.py.

TO START THE MOVIE TRAILER:

Run entertainment_center.py file, then the script will create the movie website.
